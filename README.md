# 🚗 Smart Parking System

## 📌 Project Overview

The Smart Parking System is an IoT-based parking management solution designed and simulated using Cisco Packet Tracer and Python. The system aims to improve parking efficiency by monitoring parking slot occupancy in real time and providing instant updates on slot availability.

This project demonstrates the integration of networking concepts, IoT devices, sensor-based monitoring, and automation logic to create a smart parking environment. By detecting whether parking spaces are occupied or vacant, the system helps reduce the time drivers spend searching for available parking spots.

---

## 🎯 Objectives

* Automate parking space monitoring.
* Provide real-time parking availability information.
* Simulate IoT communication between sensors and network devices.
* Demonstrate the application of networking and automation in smart city solutions.
* Improve parking management efficiency through intelligent monitoring.

## 🏗️ System Architecture

The system consists of:

* Parking slot sensors
* IoT-enabled devices
* Network infrastructure
* Monitoring and control logic
* User interface/dashboard for parking status display

### Workflow

1. A vehicle enters a parking slot.
2. The sensor detects occupancy.
3. Sensor data is transmitted through the network.
4. The system processes the information.
5. Parking status is updated in real time.
6. Users can view available and occupied parking spaces.

## 🚀 How to Run

### Cisco Packet Tracer Simulation

1. Install Cisco Packet Tracer.

3. Start the simulation mode.
4. Observe parking slot occupancy updates and device communication.
